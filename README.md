# Licious Clone

A full-stack e-commerce application replicating the core features of Licious, a premium meat delivery platform.

## Features

- Product catalog with meat categories
- Shopping cart and checkout functionality
- User authentication and profiles
- Order tracking and history
- Payment integration
- Admin dashboard for inventory management

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript (or React/Vue)
- **Backend**: Node.js/Express (or your chosen framework)
- **Database**: MongoDB/PostgreSQL
- **Authentication**: JWT/OAuth

## Getting Started

### Prerequisites
- Node.js v14+
- npm or yarn

### Installation

```bash
git clone https://github.com/noelquadras/licious-clone
cd licious-clone
npm install
```

### Running the Project

```bash
npm run dev
```

Visit `http://localhost:3000` in your browser.
